What is the Document Object Model (DOM)?

The document object model (DOM) is a programming interface that represents a web pages structure in memory as a tree of nodes and objects. This allows developers to manipulate the pages constent,style and structure using scripting languages like JavaScript

How do you select an element from the DOM?

To select an element form the DOM in JavaScript . Use methods like getElementById() to access an element by its unique ID. getElementsByClassName() to find elements with a specific class name or querySelector() to select an element based on a CSS selector , which allows for more complex targeting using combinations of IDs,classes and tag names.

What is event delegation in the context of the DOM, and why is it useful?
